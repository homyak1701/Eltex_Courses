Реализация файлового менеджера по типу mc;
***
Сборка:

1) make temp - собрать проект;
2) ./temp - запустить программу;

!Внимание!
- запускать только при развернутом терминале -
- на мониторах 3:4 отображение идет некорректно -

***
Управление:

1) стрелочки вверх-вниз для перемещения;
2) Enter, чтобы зайти в папку;
3) Tab - переключение между окнами;
