Написать эхо-сервер на сырых сокетах с протоколом UDP,IP,Ethernet
(сырой сокет реализован только на стороне клиента);

Программа будет корректно работать, если в коде
указать праввильные ip и mac адреса
***
Сборка: make
***
Запуск: 

сначала в одном тепминале: sudo ./serv
потом в другом терминале: sudo ./client
exit - для выхода
