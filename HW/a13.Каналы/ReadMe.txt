Реализация командного интерпритатора с одним ключом и конвейером
(тренеровка по работе с каналаим)
***
Управление:

введите команды bash, то есть из папки /bin/.

Данная программа позволит одной команде передать свой результат
на вод второй программы. Данная программа тестировалась только на 
примере "ls -la | grep .txt"

***
Собрать программу:

make - собрать программу;
./temp - запустить программу;
exit - команда, чтобы выйти из программы;

***
